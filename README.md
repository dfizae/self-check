# 📖 리액트+vite 실습 - useReducer 상태 훅 활용
- react+ts 공부 로그 링크 []()

## 🧩 조건
- [아이템 추가] 버튼을 만든다.
- [아이템 추가] 버튼을 클릭하면 '사과, 바나나, 오렌지'를 1개씩 추가한다.
- 아이템 3개를 추가하면 더 이상 추가할 수 없다.
- 추가한 아이템은 배열로 관리한다.
- 추가한 아이템을 `ui`, `li` 태그를 사용해 목록으로 출력한다.
- `li` 태그의 구성은 추가한 아이템과 "[삭제]" 버튼이다.
-'[삭제]' 버튼을 클릭하면 아이템을 삭제한다.
- 삭제한 아이템은 다시 추가할 수 있다.
- `useReducer` 훅을 사용한다.
